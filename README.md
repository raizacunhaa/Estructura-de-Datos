# Estructura de Datos

Este repositorio reúne los proyectos desarrollados para la materia **Estructuras de Datos**, incluyendo la implementación y resolución de los parciales mediante estructuras fundamentales como **listas doblemente enlazadas** y **árboles binarios**.

---

## ⚠️ Nota importante

Los archivos del proyecto **principal se encuentran en la rama `master`**.  
Esta rama (`main`) contiene únicamente este README para documentar el repositorio.
